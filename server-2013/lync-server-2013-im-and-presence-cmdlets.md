# Cmdlets de mensajería instantánea y presencia

 

_**Última modificación del tema:** 2016-12-08_

Los cmdlets de presencia y de mensajería instantánea permiten administrar esas características de los clientes mediante Windows PowerShell. Puede definir las directivas de presencia que se asignan a los usuarios en el ámbito global, de sitio o de usuario. También puede configurar diversas características de privacidad y mensajería instantánea.
