# Cmdlets de teléfonos y dispositivos

 

_**Última modificación del tema:** 2016-12-08_

Microsoft Lync Server 2013 proporciona una serie de cmdlets para poder administrar teléfonos y otros dispositivos de hardware. Por ejemplo, teléfonos de voz sobre IP (VoIP), teléfonos de área común (como un teléfono en un vestíbulo, una cafetería u otro lugar público), incluso teléfonos analógicos o teléfonos que no pueden ejecutar Lync Phone Edition.

## Cmdlets de teléfonos y dispositivos

Los cmdlets **CsDeviceUpdate** se usan para administrar el Servicio web de actualización de dispositivos, un componente de Lync Server con el cual los administradores distribuyen actualizaciones de firmware en teléfonos y otros dispositivos que ejecutan Lync Phone Edition.
