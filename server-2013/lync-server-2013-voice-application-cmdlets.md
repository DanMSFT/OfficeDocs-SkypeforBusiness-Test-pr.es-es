# Cmdlets de aplicaciones de voz

 

_**Última modificación del tema:** 2016-12-08_

Las aplicaciones de voz se ejecutan en el servicio de aplicación en el servidor front-end. Estas aplicaciones facilitan la administración de las llamadas entrantes y salientes al permitir el estacionamiento de llamadas, ocuparse de números no asignados, coordinar con Mensajería unificada de Exchange (UM) y ejecutar Aplicación de grupo de respuesta.
