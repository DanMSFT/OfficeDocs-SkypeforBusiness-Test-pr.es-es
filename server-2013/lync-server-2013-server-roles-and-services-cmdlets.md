# Cmdlets de servicios y roles de servidor

 

_**Última modificación del tema:** 2016-12-08_

Muchos de los componentes de Microsoft Lync Server 2013 funcionan como roles de servidor o como servicios. Lync Server 2013 se suministra con una serie de cmdlets para administrarlos.

## Cmdlets de servicios y roles de servidor

Muchas de las tareas de administración relativas a los servicios y roles de servidor se pueden efectuar en Panel de control de Lync Server. Por ejemplo, mediante Panel de control de Lync Server puede administrar la configuración del servidor de archivado, pero no el de la libreta de direcciones. No obstante, todas las tareas se pueden realizar con cmdlets desde Shell de administración de Lync Server o bien en un script; el uso de un script permite automatizar algunas tareas. A continuación se presenta una lista de cmdlets directamente relacionados con la administración de servicios y roles de servidor:
